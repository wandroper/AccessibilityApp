# ♿ Accessibility Helper App

Aplikasi Android untuk membantu pengguna dengan keterbatasan motorik.

## Fitur
- 🪟 **Floating Menu** - ikon melayang yang bisa di-drag
- ⬆⬇ **Auto Scroll** - scroll otomatis dengan interval 1.5 detik
- 🎯 **Auto Click** - deteksi & klik elemen otomatis
- 🔁 **Background Service** - tetap aktif saat app ditutup

---

## Cara Build APK via GitHub (Tanpa PC)

### Langkah 1 — Upload ke GitHub
1. Buka **github.com** di browser HP
2. Login / daftar akun GitHub (gratis)
3. Klik tombol **"+"** → **"New repository"**
4. Nama repo: `AccessibilityApp`
5. Pilih **Public** → klik **"Create repository"**

### Langkah 2 — Upload File
1. Di halaman repo kosong, klik **"uploading an existing file"**
2. Upload semua file dari ZIP ini dengan **mempertahankan struktur folder**
3. Klik **"Commit changes"**

### Langkah 3 — Tunggu Build Otomatis
1. Klik tab **"Actions"** di repo
2. Akan muncul workflow **"Build APK"** yang sedang berjalan ⏳
3. Tunggu ~3-5 menit hingga muncul ✅ hijau

### Langkah 4 — Download APK
1. Klik workflow yang sudah selesai (✅)
2. Scroll ke bawah ke bagian **"Artifacts"**
3. Klik **"AccessibilityApp-debug"** → APK terdownload!

---

## Cara Install APK di HP Android
1. Pindahkan APK ke HP
2. Buka **Pengaturan** → **Keamanan** → aktifkan **"Sumber tidak dikenal"**
3. Buka file APK → Install
4. Ikuti 3 langkah di dalam aplikasi

---

## Cara Pakai Aplikasi
1. **Izin Overlay** → tap tombol biru → izinkan
2. **Accessibility** → tap tombol hijau → cari "Accessibility Helper" → aktifkan
3. **Start** → tap tombol orange → floating ikon muncul di layar
4. **Toggle menu** → tap ikon melayang untuk expand/collapse
5. **Drag** → tahan & geser ikon ke posisi yang nyaman
